Comando de GIT

Para crear archivos desde cmd

echo hola > "index.html"

Para iniciar git

git init

Para entrar a visual studio code

code .

para ver el estatus de los archivos de mi rep de git

git status

Agregar nuestros archivos

git add .

Para guardar nuestros progreso mediante mensaje

git commit -m "Primero commit"

Historial de mis commit

git log

Para ir a un commit copiar codigo de log

git checkout bdf06c42bd34d83949f1fef9e295a9d725721100

Para ver las ramas de git

git branch

Para cambiar de ramas

git switch master

crear unas nuevas ramas

git switch -c "nombreDeLaRama"
