Comando de GIT

Para crear archivos desde cmd

echo hola > "index.html"

Para iniciar git

git init

Para entrar a visual studio code

code .

para ver el estatus de los archivos de mi rep de git

git status

Agregar nuestros archivos

git add .

Para guardar nuestros progreso mediante mensaje

git commit -m "Primero commit"

Historial de mis commit

git log
